# CAD Assets

This folder contains CAD exports and documentation for the MakerMania double worm gear mechanism.

Contents:

- `gear_draft_1.stl` — primary gear STL (please upload the full binary here if not already present).
- `gear_draft_1.md` — design notes, print settings, assembly, and test guidance.

How to add the STL to this folder:

1. Rename the attached file `Gear Wheel.stl` to `gear_draft_1.stl`.
2. Upload or copy the file into this `cad/` folder in the repository.
3. Optionally add STEP/IGES exports for parametric edits.

Recommended next steps:

- Review `gear_draft_1.stl` in your CAD viewer (Fusion 360 / FreeCAD / PrusaSlicer).
- If the model needs dimension adjustments, export a STEP and include it as `gear_draft_1.step`.
- Commit changes and push to the remote repository; include a short commit message: "Add gear draft 1 STL + docs".
